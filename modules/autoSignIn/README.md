# 自动签到模块

<div align="center">

> 提供自动完成每日签到功能，支持随机和固定模式，让你不错过每天的鸡腿奖励。

## ✨ 功能特性

### 🎯 核心功能

<table>
<tr>
<td>

- 🔄 **自动签到** - 每日自动完成签到
- 🎲 **随机模式** - 支持随机奖励签到
- 📌 **固定模式** - 支持固定奖励签到
- 🎯 **手动触发** - 支持立即执行签到
- ⚡ **智能检测** - 自动识别登录状态
- 💫 **状态提示** - 清晰的操作反馈

</td>
</tr>
</table>

### ⚙️ 可配置项

<table>
<tr>
<td>

| 配置项 | 说明 | 默认值 |
|-------|------|--------|
| 签到模式 | 设置签到奖励模式 | ❌ 禁用 |
| 立即签到 | 手动触发签到操作 | - |

**签到模式选项：**
- 禁用自动签到
- 随机签到模式
- 固定签到模式

</td>
</tr>
</table>

## 🎯 使用方法

<table>
<tr>
<td>

1. 基础设置：
   - 选择签到模式
   - 系统将在每日自动执行
   - 可随时更改签到模式

2. 手动签到：
   - 点击"立即签到"按钮
   - 等待签到结果提示
   - 查看获得的鸡腿数量

3. 状态查看：
   - 自动显示签到结果
   - 提示获得的奖励数量
   - 显示当前鸡腿总数

</td>
</tr>
</table>
